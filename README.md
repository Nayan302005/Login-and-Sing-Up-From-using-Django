\

💻 Secure & Structured Authentication System

---

## 🌟 Overview

This project is a full-stack authentication system developed using Django.
It demonstrates strong backend fundamentals, secure authentication handling, and structured MySQL database integration.

The main objective of this project is to build a clean and scalable authentication workflow using Django’s built-in authentication system.

---

## ✨ Core Features

✔️ User Registration
✔️ Secure Login & Logout
✔️ Session Management
✔️ Protected Dashboard Access
✔️ MySQL Database Connectivity
✔️ Clean & Minimal UI Design

## ⚙️ Installation Guide

### 🔹 Clone Repository

```
git clone https://github.com/yourusername/your-repo-name.git
cd your-repo-name
```

### 🔹 Create Virtual Environment

```
python -m venv venv
```

### 🔹 Activate Environment (Windows)

```
venv\Scripts\activate
```

### 🔹 Install Dependencies

```
pip install -r requirements.txt
```

### 🔹 Run Development Server

```
python manage.py runserver
```

Open browser → http://127.0.0.1:8000/
